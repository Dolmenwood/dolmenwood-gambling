# Dolmenwood Gambling Tables

A mobile-friendly, browser-based implementation of tavern gambling rules inspired by the Dolmenwood TTRPG setting.

https://dolmenwood.github.io/dolmenwood-gambling/

## Features

* **Fully Static:** Runs entirely in the browser with HTML, CSS, and JavaScript.
* **Mobile First:** Designed for phone screens with large touch targets and readable fonts.
* **Game Logic:**
    * **Cheat Mode:** Choose a modifier to rig the dice, but risk getting caught.
    * **Gambling:** Roll 2d6 against the house to win or lose stakes.
    * **Complications:** High winnings may trigger consequences from the Thieves' Guild or rivals.

## How to Run Locally

1.  Download the files (`index.html`, `styles.css`, `main.js`).
2.  Place them in a single folder.
3.  Double-click `index.html` to open it in your web browser.

## How to Publish to GitHub Pages

You can host this for free so you can access it on your phone anywhere.

### Step 1: Create the Repository
1.  Log in to [GitHub](https://github.com).
2.  Click the **+** icon in the top right and select **New repository**.
3.  Name it `dolmenwood-gambling` (or whatever you like).
4.  Check "Add a README file" (optional, but helpful).
5.  Click **Create repository**.

### Step 2: Upload Files
1.  In your new repository, click **Add file** > **Upload files**.
2.  Drag and drop your `index.html`, `styles.css`, and `main.js` files into the box.
3.  In the "Commit changes" box at the bottom, type "Initial commit".
4.  Click **Commit changes**.

### Step 3: Enable GitHub Pages
1.  Click the **Settings** tab at the top of your repository page.
2.  On the left sidebar, look for **Pages** (under the "Code and automation" section).
3.  Under **Build and deployment** > **Branch**, select `main` (or `master`) from the dropdown menu.
4.  Click **Save**.

### Step 4: Access the Game
1.  Wait about 1-2 minutes. Refresh the Settings > Pages page.
2.  A bar will appear at the top saying: "Your site is live at..."

3.  Click that URL to play the game on any device!
